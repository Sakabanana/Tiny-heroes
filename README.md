# Tiny Heroes

一款基于Python开发的俯视角动作射击游戏，支持键盘和手势控制两种操作模式。

## 🎮 游戏简介

《Tiny Heroes》是一款快节奏的动作射击游戏，玩家需要控制英雄角色，在越来越强的敌人浪潮中生存下来。游戏提供了两种独特的控制方式：传统的键盘控制和创新的摄像头手势控制，让玩家体验不同的游戏操作乐趣。

## ✨ 游戏特色

### 双模式控制系统
- **键盘控制**：WASD移动，J键射击，K键近战攻击
- **手势控制**：通过摄像头识别手部动作，右手控制移动，左手控制攻击

### 多样化的敌人类型
- **近战敌人**：快速接近玩家造成伤害
- **远程敌人**：保持距离并发射子弹
- **BOSS敌人**：强大的首领，拥有更高的生命值和特殊攻击

### 丰富的游戏机制
- 道具系统：生命恢复、攻击提升、速度提升
- 陷阱机制：需要小心规避的尖刺陷阱
- 波次系统：共15波敌人，难度逐步提升
- 得分系统：击败不同敌人获得相应分数

### 精美的视觉效果
- 渐变色彩和光影效果
- 3D立方体风格的敌人
- 粒子特效和血迹效果
- 现代化的UI界面

## 🎯 游戏规则

### 得分规则
| 敌人类型 | 得分 |
|---------|------|
| 近战敌人 | 50分 |
| 远程敌人 | 80分 |
| 陷阱 | 30分 |
| BOSS | 500分 |

### 道具效果
- **绿色道具（生命恢复）**：恢复20点生命值
- **橙色道具（攻击提升）**：攻击力+10，持续10秒
- **蓝色道具（速度提升）**：移动速度+30%，持续10秒

### 胜利条件
成功击败所有15波敌人（包括最终BOSS）即可获得胜利。

## 🛠️ 技术栈

- **游戏引擎**：Pygame 2.6.1+
- **手势识别**：MediaPipe 0.10.0+
- **图像处理**：OpenCV-Python 4.5.0+
- **数值计算**：NumPy 2.2.6

## 📦 安装与运行

### 环境要求
- Python 3.8 或更高版本
- 支持OpenCV的摄像头（手势模式需要）

### 安装步骤

1. **克隆仓库**
   ```
   git clone https://github.com/Sakabanana/Tiny-heroes.git
   cd Tiny-heroes
   ```

2. **创建虚拟环境（推荐）**
   ```
   # Windows
   python -m venv venv
   venv\Scripts\activate

   # macOS/Linux
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **安装依赖**
   ```
   pip install -r requirements.txt
   ```

4. **运行游戏**
   ```
   python main.py
   ```

## 🎮 操作说明

### 键盘模式
- **W/A/S/D**：上下左右移动
- **J**：远程射击
- **K**：近战攻击
- **ESC**：退出游戏

### 手势模式

**右手控制移动**
- 右手向右：向右移动
- 右手向左：向左移动
- 右手向上：向上移动
- 右手向下：向下移动

**左手控制攻击**
- 左手未出现：自动连续射击（约0.2秒/次）
- 左手出现：单次射击

## 🔧 常见问题

### 手势模式无法启动怎么办？
请确保：
1. 摄像头已正确连接并允许访问
2. 已安装所有依赖：`pip install opencv-python mediapipe`
3. 光线充足，手部清晰可见

### 游戏运行卡顿？
可以尝试：
1. 关闭其他占用资源的程序
2. 降低游戏分辨率（需要修改代码中的窗口尺寸）
3. 确认电脑满足最低配置要求

### 如何修改游戏难度？
可以在 `main.py` 中修改以下常量：
- `MAX_WAVES`：最大波次数（默认15）
- `ENEMY_SPEED`：敌人移动速度
- `ITEM_SPAWN_TIME`：道具生成间隔
